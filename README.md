# Além do Algoritmo

Curadoria pessoal de canais do **YouTube** e **Instagram** com conteúdo de qualidade,
organizada manualmente para evitar a dependência do algoritmo das redes sociais.

Este repositório existe para centralizar fontes que realmente valem o tempo de consumo: conteúdo claro, consistente e com valor real de aprendizado.

---

## 🎯 Objetivo

- Centralizar bons canais e perfis
- Facilitar revisitas a conteúdos relevantes
- Fugir do ruído algorítmico das redes sociais
- Manter uma curadoria pessoal simples e sustentável

---

- 📸 [Instagram](./instagram.md)
